Практическая работа #11

Организация условных переходов c if - else

Имя файла: pr11_fio.cpp, где fio - инициалы фамилии, имени, отчества. Файл результатов pr11_fio.dat (в подкаталог FIL_DAT).

Порядок выполнения работы

  1. Откройте Far. Войдите в каталог Borland на диске C:, найдите подкаталог для практических работ PROGRAM . Создайте файл pr11_fio.cpp (Shift+F4). Скопируйте приведенный ниже пример (не забудьте поменять кодировку на 866 – клавиша F8)
  2. Выполните вызов Borland C++, откройте там созданный файл и запустите (Меню Run-Run)
  3. Изучите теоретический материал и примеры программ темы
    • Операторы передачи управления.
    • Условный оператор ? :.
  4. Приступите к выполнению индивидуального задания.

Задание.

Составить блок-схему алгоритма и программу выполнения задания вашего варианта. Значения аргументов ввести с клавиатуры. Значения как аргументов, так и результатов с необходимыми пояснениями вывести на экран и в файл.

Требования:

  • Блок-схему алгоритма начертить в тетради.
  • На экран вывести вначале первую страницу - функция page_1(), затем два окна вывода:
    • окно 1 - название работы,
    • окно 2 - запрос значений аргументов, результаты выполнения работы.
  • Для построения окон использовать функцию собственной библиотеки Wind().
  • Используя операторы условных и безусловных переходов, организовать окончание работы программы на ввод символа.
  • Результаты выполнения практического задания также записать в файл в подкаталоге FIL_DAT.
  • Вывод результатов на экран сопровождать пояснениями.
  • В программе использовать ремарки (пояснения).

Пятеркой оценивается своевременно выполненная работа, если

  • блок-схема начерчена в тетради,
  • программа оформлена с соблюдением хорошего стиля программирования,
  • необходимые файлы собственных библиотек, хранящиеся в каталоге H, подключены к программе, из которой они вызываются,
  • создано заданное количество эстетически оформленных окон ввода\вывода,
  • при вычислении получены верные результаты,
  • организован вывод на экран, как входных данных - аргументов, так и выходных данных - результатов,
  • организован вывод значений в файл.

11 Известна площадь круга s1 и площадь квадрата s2. Определить: поместится ли круг в квадрат?
12 Если целое число m делится нацело на целое число n , то вывести на экран частное от деления, в противном случае вывести на экран сообщение " m на n нацело не делится" и значение остатка.
13 Определить, является ли данное целое число n нечетным четырехзначным числом.
14 Посчитать количество четных и количество нечетных чисел, среди введенных с клавиатуры чисел а , b , с , d. Вывести эти числа и их количество.

Год является високосным годом, если его номер делится на 4, за исключением тех, которые делятся на 100 и не делятся на 400. int main(void) {
    clrscr();

    FILE * f;
    #define ESC 27
    // open file
    if ((f = fopen("fil_dat\\pr11_EIN.dat", "w")) == NULL) {
        fprintf(stderr, "Error! Can't open a file...\n");
        return 1;
    }
    cout << "Hi, this program determines whether the graph of the function y = 5x^2-7x + 2 passes through a given point with coordinates (a, b).\n";
    start:
        // input variables
        cout << "Input a: ";
        cin >> a;
        cout << "Input b: ";
        cin >> b;
        // substitute a - for x and check whether this expression is equal to the number b
        if(5*a*a-7*a+2 == b) {
            cout << "[+] Point with coordinates " << "(" << a << ":" << b << ")" << " included in the function schedule y = 5x^2 - 7x + 2" << endl;
            fprintf (f, "[+] Point with coordinates a = %f\tb = %f\t included in the function schedule y = 5x^2 - 7x + 2", a, b);
        }
        // if the expression is not equal to b
        else {
            cout << "[-] Point with coordinates " << "(" << a << ":" << b << ")" << " not included in the function schedule y = 5x^2 - 7x + 2" << endl;
            fprintf (f, "[-] Point with coordinates a = %f\tb = %f\t not included in the function schedule y = 5x^2 - 7x + 2", a, b);
        }
        cout << "\nTo stop entering Esc\n";
        cout << "To continue typing, press any key\n";
        
        if ((ch = getch()) != ESC){
            goto start;
        } else {
            fclose(f);
            
            cout << "\nThank you for using my program\n";
            getch();
        }
        return 0;
}
